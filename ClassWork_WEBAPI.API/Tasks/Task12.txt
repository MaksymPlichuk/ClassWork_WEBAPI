Почати роботу над фінальним проектом
Як дз скинути посилланя на репозиторій з вашим початим проектом

На фінальну роботу потрібно написати бекенд (варіанти на вибір):
	- Для вашої фінальної роботи по React
	- Бекенд на будь яку тему (Не бібліотека). Повинно бути мінімум 3 таблиці
	- Бекенд для телеграм бота (Бот повинен містити не мало цікавого функціоналу)
	
Буде плюсом якщо будуть використані бібліотеки:
	- Quartz

	- Serilog
